Agenda
======
SYSU SS TRNG 2014 - Agenda

make 生成可执行的 Agenda

agenda.data:存储会议信息和用户信息
Log.data:存储系统日志

没登陆系统时可执行操作:
l      -    Log in Agenda by user name and password
r      -    Register an Agenda account
q      -    Quit Agenda

登录系统后可执行操作:
o      -    Log out Agenda
dc     -    Delete Agenda Account
lu     -    List All Users
cm     -    Create A Meeting
la     -    List All Meetings
las    -    List All Sponsor Meetings
lap    -    List All Participate Meetings
qm     -    Query Meeting By Title
qt     -    Query Meeting By Time Interval
dm     -    Delete Meeting By Title
da     -    Delete All Meetings
cf     -    Change user file

管理员账号:admin
管理员密码:admin




doc : architecture graph and data file format
src : header files for Agenda Project
